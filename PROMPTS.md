**Промпт 6**

*Результат P3:* Настроен Tailwind CSS, создан React Query клиент с JWT-интерцептором, страница входа, Layout с боковой навигацией по ролям, ProtectedRoute, личный кабинет сотрудника (баланс + статус), форма пожеланий с валидацией и предупреждениями о HIGH-сезоне.

## P4 — Frontend менеджер

*Результат P4:* Реализованы все страницы менеджера:
- **Дашборд** — статистика + кнопка генерации + статус-бар
- **Диаграмма Ганта** — CSS Grid, группировка Цех→Смена, фоновые сезоны, цвета статусов
- **Тепловая карта** — покрытие по неделям, зелёный/жёлтый/красный
- **Согласование** — фильтры, approve/modify с комментарием, модальное окно
- **Конфликты** — список с кодами C-01…C-05, severity
- **Структура** — CRUD цехов, смен, сотрудников
- **Календарь** — управление сезонными периодами и нормами покрытия
